# Prozesse

## Event Ablauf

1. Anfrage wird erfasst → Status: Anfrage
2. Planung läuft → Status: In Bearbeitung
3. Event bestätigt → Status: Bestätigt
4. Event durchgeführt → Archivierung
5. Debriefing wird ausgefüllt
6. Event wird archiviert

---

## Archivierung

* Debriefing ist Pflicht
* Event wird in Archiv verschoben
* keine Bearbeitung mehr möglich
* nur Admin darf löschen

---

## Benutzer

* Benutzer wird durch Systemadmin erstellt
* Initialpasswort wird vergeben
* beim ersten Login MUSS Passwort geändert werden
