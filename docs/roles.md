# Rollen & Rechte

## Leser

* Events ansehen
* Dashboard sehen
* Archiv ansehen
* PDF exportieren
* eigenes Profil bearbeiten

## Bearbeiter

* Events erstellen
* Events bearbeiten (alle)
* Events archivieren (inkl. Abschluss)
* KEIN Löschen

## Admin

* alles von Bearbeiter
* Events löschen (auch archivierte)
* Räume verwalten
* Preislisten verwalten
* Offerten & Rechnungen
* Stammdaten pflegen

## Systemadmin

* Benutzer erstellen/löschen
* Lizenzen verwalten
* Betriebe verwalten
* Systemlogs einsehen
* Backups verwalten
